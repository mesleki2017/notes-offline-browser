# notes-offline-browser

Tarayicida calisan, kurulumsuz ve offline kullanilabilen basit bir not defteri.

## Kullanim

`index.html` dosyasini tarayicida acin.

## Ozellikler

- Not ekleme, duzenleme ve silme
- Baslik, kategori, renk ve onemli isareti
- Arama, filtreleme ve siralama
- JSON olarak ice/disa aktarma
- Tarayici icinde offline calisma
